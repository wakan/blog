Remarque sur l'éditeur de markdown de dillinger

Oct 10, 2016 · 1 min read

Dillinger est un éditeur de markdown avec une interface assez responsive. Il a une prévisualisation instantanée du markdown avec un export pdf pour le donner à relire à quelq’un.

Il est lié directement avec un compte github ce qui pourrait permettre de créer des articles sur Hugo directement avec un outil convivial.

La limite que je vois actuellement, c’est qu’il faut d’abord créer le document sur github pour après l’importer dans dillinger pour qu’il se sauvegarde automatiquement sur github.
